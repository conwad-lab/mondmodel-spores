# mondmodel-spores

SPORE network colony repo for **mondmodel-prime**.

Shares operational learnings (graveyard failures, winning genomes, fitness functions) with peer colonies per the [SPORE protocol](https://github.com/conwad-lab/mondmodel/blob/main/.claude/spore/SPORE_PROTOCOL.md) — validate before integrate, attribution persists, no central authority.

## Structure

- `graveyard/` — failed patterns with autopsy, so peers avoid the same dead end
- `genome/` — winning/replicable patterns (procedures, fixes, diagnostic methods)
- `fitness/` — evaluation methods and metrics found useful
- `governance/` — institutional/process patterns (sharing disabled by default in mondmodel-prime's config)

## Origin colony

`mondmodel-prime` — Mondmodel Traveller (hotel curation) + Mondmodel Consulting (Dignami/BLK Tenet). See the origin repo's `.claude/spore/config.md` for privacy settings and thresholds.
